public enum RequestValidation

  • none: No validation
  • successCodes: Standard HTTP success codes (200,…,299)
  • successAndRedirectCodes: Standard HTTP success and redirect codes (200,…,399)
  • customCodes: Custom collection of valid HTTP codes
